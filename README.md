# Class Schedule API

## Description

This project is part of the Student Life Hub system.
It provides an API endpoint for retrieving class schedule data.

## Features

* Fetch class schedule from MockAPI
* Display subject, time, day, and room
* Built using Laravel

## API Endpoint

GET /api/schedule

## Sample Response

```json
[
  {
    "subject": "System Integration Architecture",
    "time": "10:00 AM - 1:00 PM",
    "day": "Saturday",
    "room": "Room 305"
  }
]
```

## Technologies Used

* Laravel (Backend)
* MockAPI
